Installation

Step 1

Download and install Buildify website builder


Step 2

In the Dist folder,double click on the app file *(no need of any external installation even no python is required)

Step 3

After the app is start successfully,type: 127.0.0.1:5000 in your browser to check the website


Step 4

Now to edit site according to your details,type 127.0.0.1:5000/edit/1 in a new tab to edit the site


Step 5

For editing tutorial,check out this video


Step 6

After all the editing is done,click on donwload site button to download it's HTML file


Step 7

To organize things and properly manage them,create a new folder of any name(e.g:My portfolio) and then paste the downloaded HTML file in it


Step 8

Now in the Buildify folder's Dist folder,you'll see a static folder,just copy that folder and paste it in our newly created folder(My portfolio)


Step 9

Your site is ready,just double click on the HTML file and you'll see your site,to publish your site you can host it on a free server
